Removes an existing directory as well as any files or subdirectories it contains.

```lua
os.rmdir("path")
```

### Parameters ###

`path` is the file system path to be removed.


### Return Value ###

True if successful, otherwise nil and an error message.


### Availability ###

Premake 4.0 or later.
